# Task Management Web App

Aplicație simplă de gestionare a sarcinilor (to-do list) dezvoltată cu HTML, CSS și JavaScript.

## Funcționalități

- Adăugare sarcini noi
- Ștergere sarcini
- Interfață responsive
- Design modern

## Tehnologii

- HTML5
- CSS3
- JavaScript (Vanilla)

## Utilizare

Deschideți `index.html` în browser.
